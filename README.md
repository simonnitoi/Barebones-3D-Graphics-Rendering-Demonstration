# Barebones 3D Graphics Rendering Demonstration
Renders a 3D object from scratch using just python.

## How It Works
- Uses the trigonometry to convert points from 3d space onto a 2d screen to be rendered via `tkinter`. Inspect the code for more details. *Note: Only wireframe cubes have been integrated. You change their size, location, rotation, colour, thickness (drawn), etc.*

## Setup
1. Clone the repository.
2. Set up the motion you want in the `render` funtion at the bottom.
3. Run python `main.py`.
