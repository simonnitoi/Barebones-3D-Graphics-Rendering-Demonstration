# Barebones 3D Cube Render
Renders a 3D object from scratch.

## How It Works
- Uses the trigonometric formulas to convert points from 3d space onto a 2d screen to be rendered via `tkinter`. *Note: This is still being worked on. **Positioning and sizing seem to work fine when rotation mechanic is disabled, which can be done by commenting out the `rotatePoint` calls and returning `pointsUnrotated` in `getCubePoints`.***

## Setup
1. Clone the repository.
2. Run python `main.py`.
